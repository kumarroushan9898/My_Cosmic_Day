🌌 MyCosmicDay
MyCosmicDay is a lightweight, framework-free web app that lets you explore NASA’s Astronomy Picture of the Day (APOD) for any date — including your birthday 🎂✨
Built using vanilla HTML, CSS, and JavaScript, it delivers a smooth and elegant space experience without any heavy dependencies.

🚀 Features
* 📅 Date PickerChoose any date to view NASA’s APOD.
* 🎂 Birthday ExplorerDiscover what the universe looked like on your birth date.
* 🖼️ Full-Screen Image ModalClick images for an immersive viewing experience.
* 🔍 HD ToggleSwitch between standard and HD image quality.
* 🌙 Dark Space ThemeStylish UI with glowing accents and cosmic visuals.
* ⚡ Fast & LightweightNo frameworks — just pure JavaScript.

🛠️ Tech Stack
* HTML5
* CSS
* JavaScript 
* NASA APOD API

🌐 API Reference
This project uses NASA’s official API:
👉 https://api.nasa.gov/
Example Request
https://api.nasa.gov/planetary/apod?api_key=YOUR_API_KEY&date=YYYY-MM-DD

🔑 Getting an API Key
1. Visit https://api.nasa.gov/
2. Sign up for a free API key
3. Replace YOUR_API_KEY in your code

🧠 How It Works
1. User selects a date
2. JavaScript sends a request to NASA APOD API
3. API returns image, title, and description
4. Data is dynamically rendered on the page
